# Expense Tracker

A simple and interactive Expense Tracker web application built using HTML, CSS, and JavaScript. The application helps users manage their personal finances by setting a budget, recording expenses, monitoring spending, and tracking the remaining balance in real time.

## Features

* Set a total budget
* Add expense items with name and cost
* Automatically calculate total expenses
* Display remaining balance
* Delete expenses individually
* Data persistence using Local Storage
* Responsive and clean user interface
* Real-time updates without page refresh

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Browser Local Storage


## How It Works

1. Enter your total budget.
2. Add expense items and their costs.
3. The application automatically updates:

   * Total Budget
   * Total Expenses
   * Remaining Balance
4. Delete any expense when needed.
5. All data is saved in Local Storage and remains available after refreshing the page.
